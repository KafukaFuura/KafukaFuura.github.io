# 保界方法简介

考虑守恒律方程：

ut+∇⋅F(u)=0,u(x,0)=u0(x)(1)u_t+\nabla \cdot F(u)=0,\qquad u(x,0)=u_0(x) \tag{1}ut​+∇⋅F(u)=0,u(x,0)=u0​(x)(1)

假设这个问题的精确解有一个 convex invariant region G，也即

u(⋅,0)∈G→u(⋅,t)∈G∀t>0(2)u(\cdot,0)\in G\rightarrow u(\cdot,t)\in G\quad \forall t>0\tag{2}u(⋅,0)∈G→u(⋅,t)∈G∀t>0(2)

比较简单的例子就是一维标量守恒律方程，他的真实解就是初始解的值传递，而上面的性质也就退化为了极值原理。更一般的，我们的方程都是用来刻画物理规律的，所以大部分的方程也就自然而然地满足上面的规律，而我们是想通过让我们的数值解也满足相应的规律，来更精准地刻画真实解，保界的数值格式也就因此诞生，在这里我将记录斜率限制器和通量限制器两种方法，并且介绍吴老师的 GQL 方法。

# 一阶保界格式

我们考虑如下的一维守恒律方程，保界此时退化为保极值原理

ut+f(u)x=0(3)u_t+f(u)_x=0 \tag{3}ut​+f(u)x​=0(3)

他的单调格式可以写为下列形式

ujn+1=Hλ(uj−1n,ujn,uj+1n)=ujn−λ[h(ujn,uj+1n)−h(uj−1n,ujn)](4)u_j^{n+1}=H_\lambda(u_{j-1}^n,u_j^n,u_{j+1}^n)\\ =u_j^n-\lambda[h(u_j^n,u_{j+1}^n)-h(u_{j-1}^n,u_j^n)]\tag{4}ujn+1​=Hλ​(uj−1n​,ujn​,uj+1n​)=ujn​−λ[h(ujn​,uj+1n​)−h(uj−1n​,ujn​)](4)

其中λ=ΔtΔx\lambda =\frac{\Delta t}{\Delta x}λ=ΔxΔt​，而 HλH_{\lambda}Hλ​ 满足Hλ(↑,↑,↑)H_\lambda (\uparrow,\uparrow,\uparrow)Hλ​(↑,↑,↑) ，且满足适合的 CFL 条件λ≤λ0\lambda \leq \lambda_0λ≤λ0​ 。

由于Hλ(c,c,c)=cH_\lambda (c,c,c)=cHλ​(c,c,c)=c 成立，我们可以推出：

如果m≤uj−1n,ujn,uj+1n≤Mm\leq u_{j-1}^n,u_j^n,u_{j+1}^n\leq Mm≤uj−1n​,ujn​,uj+1n​≤M

根据单调性可以直接推出

m=Hλ(m,m,m)≤ujn+1≤Hλ(M,M,M)=M(5)m=H_\lambda (m,m,m)\leq u_j^{n+1}\leq H_\lambda (M,M,M)=M\tag{5}m=Hλ​(m,m,m)≤ujn+1​≤Hλ​(M,M,M)=M(5)

# 斜率限制器：适用于有限体积与 DG 格式

# GAUSS-LOBATTO 积分回顾

在学习 Gauss-Legendre 积分时，在不加限制时，我们可以自由地选取用于积分估计的点的位置，以及相应点的权重，自然地，当选取 n 个点时，最高可以达到 2n-1 的代数精度。但有时我们会对点的位置有额外的要求，例如在构建保界限制器时用到的 Gauss-Lobatto 积分，我们需要积分的端点值也要取到，这时精度就会掉两阶，即 2n-3 阶。这个网站可以查到 [−1,1] 上的 Gauss-Lobatto 积分点和权重：Nodes and Weights of Gauss-Lobatto Calculator

# 保界限制器的思想起源

对于一个高阶线性格式而言，他并不能满足极值原理（Godunov 定理），所以我们应当寻求非线性格式。

一般地，我们设高阶格式有如下格式：

u‾jn+1=G(u‾jn,uj−1−,uj−1+,uj+1−,uj+1+)=u‾jn−λ[h(uj+1−，uj+1+)−h(uj−1−,uj−1+)](6)\overline{u}_j^{n+1}=G(\overline{u}_j^n,u_{j-1}^-,u_{j-1}^+,u_{j+1}^-,u_{j+1}^+)\\ =\overline{u}_j^n-\lambda[h(u_{j+1}^-，u_{j+1}^+)-h(u_{j-1}^-,u_{j-1}^+)]\tag{6}ujn+1​=G(ujn​,uj−1−​,uj−1+​,uj+1−​,uj+1+​)=ujn​−λ[h(uj+1−​，uj+1+​)−h(uj−1−​,uj−1+​)](6)

其中 h(↑,↓)h(\uparrow,\downarrow)h(↑,↓) ，因此，不加其他的分析与条件，我们可以得到 G(↑,↑,↓,↓,↑)G(\uparrow,\uparrow,\downarrow,\downarrow,\uparrow)G(↑,↑,↓,↓,↑) 。

为了让这个格式单调，我们有一种比较朴素的想法，就是用其他变量控制第三四个位置的变量，而用来控制这两个变量的就是 u‾jn\overline{u}_j^nujn​ 。
